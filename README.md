# Projet Data Mining - Segmentation Client (RFM & K-Means)

Ce projet de Data Mining implémente une segmentation client en utilisant le modèle RFM (Récence, Fréquence, Montant) et l'algorithme K-Means sur un jeu de données de commerce de détail.

## 📁 Structure du projet

- `notebook/TP_DataMining_Probleme_9.ipynb` : Le notebook contenant le code complet de l'analyse, la préparation des données et le clustering.
- `data/online_retail.csv` : Le jeu de données utilisé.
- `requirements.txt` : Les librairies nécessaires.

## 🚀 Comment exécuter

1. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

2. Lancez Jupyter Notebook :
   ```bash
   jupyter notebook
   ```

3. Ouvrez le fichier `notebook/TP_DataMining_Probleme_9.ipynb` et exécutez les cellules.

## 📊 Objectif

L'objectif est d'identifier différents profils de clients (comme les "Champions" ou les "Clients à risque") grâce aux données d'achat, afin de proposer des recommandations marketing adaptées.
